# 07. Экраны каталога

Главная, категории, список товаров, карточка товара, поиск, профиль продавца.

---

## 1. Главная (HomeScreen)

**Маршрут:** `/(tabs)/index`
**API:** `GET /catalog/home`
**Auth:** Нет

### Layout

```
┌─────────────────────────────────┐
│ 🔍 Поиск товаров...            │  ← SearchBar (тап → /search)
├─────────────────────────────────┤
│                                 │
│  ┌──────┐┌──────┐┌──────┐     │  ← Категории (горизонтальный скролл)
│  │  📱  ││  👕  ││  🏠  │ ... │
│  │Элект.││Одежда││ Дом  │     │
│  └──────┘└──────┘└──────┘     │
│                                 │
│  Новинки                    →  │  ← Секция с link "Все"
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐ │
│  │    │ │    │ │    │ │    │  │  ← Горизонтальный скролл
│  │ $$ │ │ $$ │ │ $$ │ │ $$ │  │     ProductCard (compact)
│  └────┘ └────┘ └────┘ └────┘  │
│                                 │
│  Популярные                     │
│  ┌─────────┐ ┌─────────┐      │  ← Grid 2 колонки
│  │         │ │         │      │     ProductCard (standard)
│  │  $99    │ │  $49    │      │
│  │  ★ 4.5  │ │  ★ 4.2  │      │
│  └─────────┘ └─────────┘      │
│  ┌─────────┐ ┌─────────┐      │
│  │         │ │         │      │
│  └─────────┘ └─────────┘      │
│        ...infinite scroll...    │
└─────────────────────────────────┘
```

### Поведение

- **Pull-to-refresh** — перезагрузить данные home
- **SearchBar** — по тапу навигация на `/search` (не inline поиск)
- **Категории** — горизонтальный ScrollView, тап → `/category/[id]`
- **Новинки** — горизонтальный FlatList, "Все" → `/products?sort_by=published_at`
- **Популярные** — вертикальный grid (2 колонки), infinite scroll через `GET /catalog/products?sort_by=published_at`
- **Skeleton loading** — показать skeleton карточек при первой загрузке

### Компоненты

- `SearchBar` — статичный (не editable), навигация на search
- `CategoryGrid` — горизонтальные карточки категорий (иконка + название)
- `ProductCarousel` — горизонтальный список товаров
- `ProductGrid` — вертикальный grid (FlashList для производительности)
- `ProductCard` — карточка товара (фото, название, цена, рейтинг, бейджи)

### Поведение

- **Фото галерея** — горизонтальный swipe, pinch-to-zoom, page indicator
- **Sticky кнопка "В корзину"** — всегда видна внизу экрана
- **Add to Cart** — haptic feedback (medium impact), toast "Добавлено в корзину", обновить badge на табе
- **Описание** — collapsible (3 строки → "Показать полностью")
- **Отзывы** — показать 2–3 первых, "Все отзывы →" ведёт к ReviewList
- **Компания** — тап на блок → `/seller/[id]`
- **Share** — нативный share sheet с deep link
- **Аналитика:** отправить `ProductViewed` при открытии, `ProductPhotoViewed` при свайпе фото, `AddToCartClicked` при добавлении

### Компоненты

- `ProductGallery` — swipeable image carousel с zoom
- `ProductBadges` — горизонтальный ряд бейджей
- `StarRating` — звёзды рейтинга (read-only)
- `SellerCard` — блок продавца (logo, name, premium badge)
- `ReviewPreview` — сокращённый список отзывов
- `ProductCarousel` — похожие товары
- `StickyAddToCart` — sticky bottom bar с кнопкой и ценой

## 5. Поиск (SearchScreen)

**Маршрут:** `/search`
**API:** `GET /catalog/products/search?q=...`
**Auth:** Нет

### Layout

```
┌─────────────────────────────────┐
│  ← [🔍 поисковый запрос...  ] │  ← Auto-focus, keyboard open
├─────────────────────────────────┤
│  (при пустом запросе)          │
│  Недавние поиски               │
│  laptop  ×                     │  ← История (локальная, MMKV)
│  phone case  ×                 │
│  headphones  ×                 │
│                                 │
│  (при вводе — debounce 300ms) │
│  ┌─────────┐ ┌─────────┐     │
│  │ result  │ │ result  │     │
│  └─────────┘ └─────────┘     │
│  ...                           │
│                                 │
│  Фильтры ▼  Сортировка ▼     │  ← Chips, тап → bottom sheet
│                                 │
│  Найдено: 42 товара            │
│  ┌─────────┐ ┌─────────┐     │
│  │         │ │         │     │
│  └─────────┘ └─────────┘     │
│  ...infinite scroll...         │
└─────────────────────────────────┘
```

### Поведение

- Auto-focus на input при открытии
- **Debounce** — 300ms после остановки ввода
- **История поиска** — последние 10 запросов (MMKV), кнопка очистки
- **Результаты** — grid 2 колонки, infinite scroll
- **Фильтры** — те же, что на CategoryProducts (цена, рейтинг)
- **Empty state** — "Ничего не найдено" с иллюстрацией
- **Аналитика:** `SearchPerformed` при выполнении поиска

## 7. Общие паттерны экранов

### Infinite Scroll

```typescript
// FlashList с infinite query
<FlashList
  data={pages.flatMap(p => p.data)}
  renderItem={({ item }) => <ProductCard product={item} />}
  onEndReached={() => {
    if (hasNextPage && !isFetchingNextPage) {
      fetchNextPage();
    }
  }}
  onEndReachedThreshold={0.5}
  ListFooterComponent={isFetchingNextPage ? <LoadingSpinner /> : null}
  estimatedItemSize={250}
  numColumns={2}
/>
```

### Pull-to-Refresh

```typescript
<FlashList
  refreshControl={
    <RefreshControl
      refreshing={isRefreshing}
      onRefresh={() => refetch()}
      tintColor={theme.colors.brandPrimary}
    />
  }
/>
```

┌─────────────────────────────────┐

│    параметры поиска             │
│                                 │
│    [ Сбросить фильтры ]        │  ← secondary button
