## 2. Вход (LoginScreen)

**Маршрут:** `/(auth)/login`
**API:** `POST /auth/login`

### Layout

```
┌─────────────────────────────────┐
│  ←                              │
├─────────────────────────────────┤
│                                 │
│  Ninhao                        │  ← Logo / brand name
│                                 │
│  Вход                          │
│                                 │
│  Email                         │
│  ┌──────────────────────────┐  │
│  │ email@example.com        │  │
│  └──────────────────────────┘  │
│                                 │
│  Пароль                        │
│  ┌──────────────────────────┐  │
│  │ ••••••••            👁  │  │  ← Toggle visibility
│  └──────────────────────────┘  │
│  Забыли пароль?            →   │
│                                 │
│  [         Войти             ] │  ← primary, full width
│                                 │
│  ─── или ───                   │
│                                 │
│  [🔐 Войти с Face ID       ]  │  ← Если biometric enabled
│                                 │
│  Нет аккаунта?                 │
│  Зарегистрироваться        →   │
│                                 │
└─────────────────────────────────┘
```

### Поведение

- Валидация на клиенте: email format, password min 1 char
- При submit: loading state на кнопке
- При 422 → показать ошибки полей
- При 401 → "Неверный email или пароль"
- При успехе:
  1. Сохранить token
  2. Предложить включить биометрию (если первый вход и устройство поддерживает)
  3. Navigate back / to redirect URL
- Keyboard avoiding: экран скроллится при показе клавиатуры

## 4. Адреса доставки (AddressesScreen)

**Маршрут:** `/addresses`
**API:** `GET /addresses`, `POST`, `PUT`, `DELETE`

- **При редактировании** — pre-fill все поля из существующего адреса
- **Валидация** — на клиенте (required fields) + серверная (422)

│                                 │

### Chat-like layout

- Сообщения пользователя — выровнены вправо (brand-primary bubble)
- Сообщения поддержки — выровнены влево (bg-secondary bubble)
- Только `visibility: external` сообщения показывать
